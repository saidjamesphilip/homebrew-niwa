# homebrew-niwa

Homebrew tap for [Niwa](https://github.com/saidjamesphilip/Niwa) — a macOS menu bar productivity app.

## Install

```bash
brew tap saidjamesphilip/niwa
brew install niwa
```

## Update

```bash
brew upgrade niwa
```

## Uninstall

```bash
brew uninstall niwa
brew untap saidjamesphilip/niwa
```
